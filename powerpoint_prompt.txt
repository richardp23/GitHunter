This challenge incorporates today‚Äôs current LLM offerings into Microsoft‚Äôs PowerPoint tool. You are tasked with utilizing development tools provided by Microsoft for PowerPoint, using Typescript or Javascript.
Challenge Overview
Build a Microsoft PowerPoint add-in that lets users use AI to create and enhance slides inside PowerPoint. The add-in should support generating slide content from prompts, transforming rough notes into well-structured slides, and doing lightweight research that can be cited or summarized into slide-ready bullets. You can use any LLM provider. This only needs to run in a development environment.

Baseline
PowerPoint add-in UI (task pane) with prompt input + output preview
Insert AI-generated content onto the current slide (title, bullets, speaker notes, images/diagrams optional)
At least one ‚Äúresearch‚Äù workflow (e.g., summarize a topic, compare options, generate pros/cons, provide sources)
Stretch ideas (pick what‚Äôs cool)

‚ÄúMake it consistent‚Äù: align terminology, tense, voice across a deck
Multi-slide mode: generate an outline and create a 5‚Äì10 slide mini-deck
Anything else you think would be cool!

Making LLM API Requests
Calling an LLM provider API directly from a PowerPoint add-in task pane is blocked by browser CORS policies and is a security risk for API keys. The recommended solution is to use a backend proxy service to forward requests securely; refer to the section titled Handling CORS for implementation details.

LLM Prompting
LLMs behave much better when you tell them the slide format explicitly.

For example: 

Create content for a PowerPoint slide.

Topic: Cloud Security Basics

Return:
- A short slide title (max 8 words)
- 4‚Äì6 bullet points (1 line each, concise)
- Optional speaker notes (2‚Äì3 sentences)

Tone: professional, clear, non-marketing
